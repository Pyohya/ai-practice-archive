# AI 학습 아카이브

공부하면서 만들어본 Notebook, 코드, 데이터셋들을 가볍게 모아둔 공간입니다. 편하게 둘러보고 필요한 건 마음껏 가져가세요!

---

## 폴더 구조

```
notebooks/  # Colab 노트북
datasets/   # 예제 데이터(소형)
scripts/    # 실험·유틸 스크립트
notes/      # 논문·강의 정리 Markdown
```

---

## 이렇게 활용해 보세요

- **노트북**으로 아이디어를 빠르게 실험해 보기
- **데이터셋**으로 소형 AI 학습에 가볍게 사용하기 
- **스크립트**로 AI 예제 학습하기
- **notes** 폴더에서 핵심 개념 & 논문 요약 한눈에 확인하기
